# Luiswap

Uniswap v2 clone

### Tech Stack:

- Hardhat
- Solidity
- TypeScript

### Installation

```shell
git clone https://github.com/llopez/luiswap
cd luiswap
yarn install
```

### Testing

```shell
yarn test
```

### Development

```shell
# start local blockchain
yarn chain
# deploy contracts to local blockchain
yarn local:deploy
# start local web3 console
yarn local:console
```
